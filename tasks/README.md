# Задачи и чек-листы

Здесь хранятся этапы и конкретные задачи по подготовке Уральского бёрна.

| Файл | Описание |
|------|----------|
| [phases.md](phases.md) | Крупные фазы от старта до пост-события |

По мере работы можно добавлять:

- `checklist-venue.md` — чек-лист по выбору и договору по месту
- `checklist-legal.md` — разрешения и документы
- `checklist-event-day.md` — что проверить в день открытия
- и т.п.

Задачи из phases.md можно дублировать в трекере (Trello, Notion, таблицы) — этот репозиторий остаётся «документацией и планом».
