# Главная страница Уральского бёрна

Вёрстка в стиле сайта PLAMEN (Черногория): тёмная тема, герой с обратным отсчётом, блоки «Что такое Burning Man», 10 принципов, формы участия, как присоединиться.

## Как открыть

Просто откройте `index.html` в браузере или поднимите локальный сервер, например:

```bash
# Python
python -m http.server 8000

# или npx
npx serve .
```

Затем откройте http://localhost:8000

## Что подправить под себя

- **Дата обратного отсчёта** — в файле `countdown.js` переменная `TARGET_DATE`. Сейчас стоит 1 июля 2026.
- **Текст дат и места** — в `index.html` блок `.hero-dates`: замените на точные даты и локацию, когда будут известны.
- **Ссылки** — кнопка «Создать Burn-профиль», ссылки в футере (телеграм, инстаграм, форма кемпа) пока ведут на `#`. Замените на реальные URL, когда появятся страницы регистрации и чаты.

Контент принципов и разделов взят из ваших документов и адаптирован под Уральский бёрн.
