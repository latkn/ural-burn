# Уральская айдентика: фестивальные элементы и промпты для ИИ

Документ для прокачки визуальной идентичности «Уральского бёрна»: идеи фестивального декора с уральским колоритом и готовые промпты для генерации визуализаций в Midjourney, DALL·E, Stable Diffusion, Kandinsky и др.

---

## Как пользоваться промптами

- **Русские промпты** — подходят для Kandinsky, ruDALL-E и других русскоязычных моделей.
- **Английские промпты** — для Midjourney, DALL·E 3, Stable Diffusion; при необходимости добавляйте `--ar 16:9`, `--v 6` и т.п.
- В квадратных скобках указаны **переменные**: подставляйте свои варианты (например, цвет фона, размер, текст).
- Для единого стиля в конце промпта добавляйте **стилевой суффикс** из раздела «Единый стилевой код» ниже.

## 1. Уральская айдентика: что подчеркивать

### 1.1 Визуальные символы и коды

| Источник | Что использовать в декоре |
|----------|----------------------------|
| **Уральские горы** | Силуэты хребтов, каменные гряды, «каменный пояс», острые и пологие вершины |
| **Шигирский идол** | Древнейшая деревянная скульптура мира; геометрический орнамент, схематические лики, вертикальная стела |
| **Пермский звериный стиль** | Медведь, лось, птица, змей; солярные знаки (спирали, круги, кресты); трёхъярусное мироустройство (низ — вода, середина — звери/люди, верх — птицы/солнце) |
| **Каслинское литьё** | Чугун, чёрный силуэт, чёткий графический контур, «голландская сажа» |
| **Златоустовская гравюра** | Сталь, травление, орнамент на металле, охотничьи/природные сюжеты |
| **Тагильская роспись** | Цветы, ягоды, листья; маховое письмо; чёрный/бордовый/зелёный фон, золотой орнамент |
| **Уральский паттерн** | Фольклорно-промышленный узор (традиционный орнамент + индустриальная графика) |
| **Башкирский/марийский орнамент** | Ромбы, солярные розетки, зигзаги, кускар (рога), геометрические бордюры |
| **Уральские самоцветы** | Малахит (зелёные полосы, бархатистая текстура), изумруд, аметист, топаз — для палитры и узоров |
| **Природа Урала** | Тайга (ель, сосна, лиственница), каменные реки, скальные останцы, глинистые обрывы |

### 1.2 Цветовая палитра

- **Природные**: тёмно-зелёный (тайга), бронзово-рыжий (сосна, осень), красная глина, серый камень, чёрный (чугун, сажа).
- **Промыслы**: чёрный + золото (тагильский поднос, каслинское литьё), малахитово-зелёный, бордовый, тёмно-синий (сталь).
- **Мифология/орнамент**: охра, умбра, белый (солярные знаки), тёмно-синий (ночь/нижний мир).

### 1.3 Настроение

Суровый, надёжный, «опорный край»; сплав Европы и Азии; огонь и металл; тайга и камни; древность (идол, звериный стиль) + промышленность (заводы, литьё, гравюра).

## 2. Фестивальные элементы: идеи

### 2.1 Флаги на столбах

- Флаги с силуэтом Уральских гор (одна или цепь вершин).
- Орнамент в стиле пермского звериного стиля или башкирского/марийского геометрического узора.
- Упрощённые мотивы Шигирского идола (лики, геометрия) на прямоугольном полотнище.
- Тагильская роспись: цветы и ягоды на тёмном фоне по краю флага, центр — логотип/символ.
- «Каменная река» — полоса камней/орнамент в виде гальки/окатанных камней.
- Чередование флагов: горы / звериный стиль / орнамент / малахитовый узор — для ритма вдоль аллеи.

### 2.2 Баннеры и растяжки

- Горизонтальные баннеры с панорамой хребта и тайги.
- Текстовые растяжки в шрифте с «горным» характером (засечки, острые формы) + орнамент по краям.
- Баннер «вход» с аркой из силуэтов гор и солярным знаком сверху.

### 2.3 Входные арки и тотемы

- Арка в виде двух стилизованных Шигирских идолов или «ворот» из гор.
- Тотемы с ликами в духе идола + звериные мотивы (медведь, лось, птица).
- Арка из «каслинского» чугунного кружева (силуэт) или из веток/коряг с орнаментом.

### 2.4 Указатели и маркеры территории

- Столбики-указатели с силуэтом горы наверху и названием зоны.
- Каменные пирамидки (турники) с нанесённым орнаментом или символом.
- Деревянные таблички в стиле пермского звериного стиля (лось, медведь, птица) для обозначения направлений.

### 2.5 Световые объекты

- Подсвеченные силуэты гор на тканевых экранах.
- Фонари с прорезным орнаментом (звериный стиль или геометрия).
- Огненные чаши с «каслинским» орнаментом по ободу.

### 2.6 Инсталляции и арт-объекты

- Уменьшенная стилизация Шигирского идола (дерево или фанера) для фото-зоны.
- Скульптура медведя/лося/птицы в пермском зверином стиле (плоскостная или объёмная).
- «Каменная река» из раскрашенных камней или подписанных участниками камней.
- Колесо/мандала с солярными знаками и тремя ярусами (низ — вода, середина — звери, верх — солнце).

### 2.7 Мелкий декор

- Гирлянды из флажков с орнаментом (ромбы, солярные знаки, малахитовые полосы).
- Бантинг с тагильскими цветами или геометрией по краю.
- Подвесные мобили: деревянные фигурки птиц/зверей в зверином стиле.
- Тканевые полосы на деревьях с орнаментом и цветами тайги.

## 7. Советы по генерации в разных сервисах

- **Midjourney**: добавляйте `--ar 2:3` для вертикальных флагов, `--ar 3:1` для баннеров; для единого стиля используйте `--sref` с одной сгенерированной картинкой.
- **DALL·E / ChatGPT**: запросы на английском часто дают стабильнее результат; можно просить «in the style of screen print» или «flat design» для графичных флагов.
- **Kandinsky / ruDALL-E**: русские промпты работают хорошо; для орнаментов уточняйте «геометрический орнамент», «без фотореализма», чтобы избежать шума.
