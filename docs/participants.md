## Правила для участников

*Чек-лист правил, которые каждый подтверждает при регистрации:*

- [ ] Следование принципам события (кратко перечислить)
- [ ] Leave No Trace — вывезти весь мусор и вещи
- [ ] Самодостаточность — вода, еда, укрытие, аптечка
- [ ] Запрет на коммерцию и рекламу на территории
- [ ] Уважение к другим и согласие (consent)
- [ ] Огонь и безопасность (где можно жечь, что нельзя)
- [ ] *Добавить по необходимости*
